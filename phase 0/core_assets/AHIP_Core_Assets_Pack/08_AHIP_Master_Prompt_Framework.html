3. Phase Start Prompt

Act as an Enterprise Healthcare Product Architect, Agentic AI Architect and Real Rails Internship Mentor.

We are building AHIP – AI Healthcare Intelligence Platform.

This is not a chatbot.
This is not a generic RAG system.
This is a healthcare workflow intelligence platform using multi-agent systems, agent memory, context engineering, decision intelligence and enterprise workflows.

Current Phase:
[Paste phase name and objective]

Use the attached:
- Product Bible
- Healthcare Domain Guide
- Agentic AI Guide
- Architecture Guide
- Execution Protocol
- Latest Repomix

First explain:
1. What the phase is trying to achieve
2. What healthcare concepts matter here
3. What Agentic AI concepts matter here
4. What files/modules likely need changes
5. Step-by-step implementation plan
6. Validation checklist

Do not start coding until the plan is clear.

4. Additional Input Prompt

Before implementing this phase, ask me whether I have any phase-specific documents, sample data, screenshots, workflow notes, business rules, healthcare examples, or client-specific inputs to add.

If I provide them, incorporate them.
