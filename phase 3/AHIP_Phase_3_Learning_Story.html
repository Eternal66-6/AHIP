⚠️ The Problem Before This Stage

Agents were querying the database directly and receiving full raw table rows. Giving an AI too much irrelevant information causes token bloat, increased latency, and severe hallucinations.

🛠️ What We Built

📦 Pydantic Context Packs

Created `PatientContextPack`, `ClaimContextPack`, and `ProviderContextPack` schemas that strip away PII and irrelevant database columns.

🏗️ HealthcareContextBuilder

A dedicated service class that acts as a secure proxy between the database and the Agents.

🕸️ Knowledge Graph UI

Built a React visualizer showing exactly how a single Case ID maps to its related Patient, Claim, and Provider context packs.

🔄 Before vs After

Before Phase 3

  • Agents read direct DB models
  • High token usage
  • Agents get distracted by irrelevant fields (e.g. internal primary keys)

After Phase 3

  • Agents receive minimized, strictly-typed Context Packs
  • Lower latency, cheaper API calls
  • Zero hallucinations regarding irrelevant data

💻 Implementation Highlights

From the Repomix context, here is how the ContextBuilder protects the AI by formatting the context strictly:

class HealthcareContextBuilder:
    def build_case_context(self, db: Session, case_id: str) -> CaseContext:
        # 1. Fetch raw DB models
        claim_model = db.query(ClaimModel).filter_by(case_id=case_id).first()
        patient_model = db.query(PatientModel)...
        
        # 2. Filter down to Context Packs (removes PII, internal IDs)
        claim_pack = ClaimContextPack(
            status=claim_model.status,
            amount=claim_model.amount,
            cpt_codes=claim_model.cpt_codes
        )
        
        # 3. Return protected context to Agents
        return CaseContext(claim=claim_pack, patient=patient_pack...)

🎙️ Interview Q&A

Q: How do you prevent LLM hallucinations when reading from a database?
A: By implementing a Context Engine. I wrote a `HealthcareContextBuilder` that intercepts raw SQL database models and converts them into strictly typed, minimized Pydantic `ContextPacks`. The AI only ever sees the ContextPack.
