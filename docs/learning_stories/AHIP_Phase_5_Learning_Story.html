🖼️ Explainable AI UI

💻 Implementation Highlights

From the Repomix context, here is how the Priority Queue sorts Risk in Python:

@router.get("/priority-queue")
def get_priority_queue(db: Session = Depends(get_db)):
    logs = db.query(AgentExecutionLog).filter(AgentExecutionLog.agent_name == "Consolidator Agent").all()
    
    # Sort logic: High risk items return 0, Low risk return 1, ensuring High Risk floats to the top
    def sort_key(log):
        is_high_risk = "Senior Claims Analyst" in (log.related_workflow_event or "") 
        return (0 if is_high_risk else 1, log.created_at)
        
    sorted_logs = sorted(unique_logs, key=sort_key)
    return sorted_logs
