RealRails Learning Story Factory v1 Pack

Files included:
1. RealRails_LearningStoryFactory_v1.skill.md
2. RealRails_LearningStoryFactory_Intern_Prompt.md
3. RealRails_LearningStoryFactory_Group_Post.html

How to use:
- After every phase/stage, intern attaches Product Bible, latest Repomix, completion notes and evidence.
- Intern uses the prompt with the skill.
- AI generates one animated self-contained HTML story.
- Intern uses the HTML to understand and explain the completed stage.

Purpose:
This improves communication, interview readiness, demo quality and true understanding.
